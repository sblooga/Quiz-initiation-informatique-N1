Quiz d'initiation à l'informatique

Un parcours ludique pour découvrir les bases de l'informatique en toute simplicité.

Bienvenue

Ce quiz a été conçu pour accompagner les seniors dans la découverte du monde numérique. Chaque question est accompagnée d'explications claires afin de faciliter l'apprentissage.

Objectifs du quiz

  • Comprendre le vocabulaire informatique de base.
  • Apprendre à utiliser les outils du quotidien numérique.
  • Gagner en confiance dans l'utilisation de l'ordinateur.
